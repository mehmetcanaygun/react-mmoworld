# MMOWorld

A React app that is created w/ TypeScript & TailwindCSS to display MMO games using [MMO API](https://rapidapi.com/digiwalls/api/mmo-games)

**To run the project**

`npm install`

`npm start`

`npm run build`

**PS**: Make sure to use your own RapidAPI in 'src/services/index.ts'.
